# HR Analytics Dashboard

## Project Overview

This project analyzes employee workforce data using SQL, Excel, and Power BI. The objective is to understand employee demographics, salary distribution, performance trends, and attrition patterns to support HR decision-making.

## Project Workflow

1. Analyzed employee data using SQL queries.
2. Exported query results to Excel for data organization and validation.
3. Used Excel for data preparation and intermediate analysis.
4. Built an interactive dashboard in Power BI.
5. Generated business insights related to employee performance and attrition.

## Tools Used

* SQL
* Microsoft Excel
* Power BI
* DAX

## Key Metrics

* Total Employees
* Average Salary
* Average Performance Score
* Attrition Count
* Attrition Rate
* Department-wise Employee Distribution

## Dashboard Features

### Employee Overview

Provides a summary of workforce demographics and employee distribution.

### Salary Analysis

Analyzes salary trends across departments and job roles.

### Performance Analysis

Evaluates employee performance scores and department-wise performance.

### Attrition Analysis

Identifies employee attrition patterns across departments and locations.

### Department Insights

Compares workforce performance and salary metrics across departments.

## Key Insights

* Finance department recorded the highest average performance score.
* IT and Finance departments showed higher average salaries compared to other departments.
* Employee attrition was concentrated in specific departments.
* Departments with higher performance scores generally showed lower attrition rates.
* Workforce distribution varied significantly across departments.

## Business Value

This dashboard helps HR teams:

* Monitor employee performance
* Track attrition trends
* Identify high-performing departments
* Support workforce planning
* Improve employee retention strategies
